# Takin Asli

این پروژه یک وب‌سایت استاتیک چندصفحه‌ای و چندزبانه است که برای نمایش خدمات، اطلاعات شرکت، صفحات آموزشی/تحقیقاتی و مسیرهای تعامل کاربر طراحی شده است. در نسخه‌ی فعلی، ساختار سایت بر پایه‌ی فایل‌های HTML آماده شده و برای توسعه‌ی سریع و قابل‌توسعه طراحی گردیده است.

## این برنامه چه کار می‌کند؟

این پروژه به‌عنوان یک قالب یا پایه‌ی وب‌سایت برای ارائه‌ی محتوا و خدمات عمل می‌کند. در این نسخه می‌توان از صفحات زیر استفاده کرد:

- صفحه‌ی اصلی و معرفی خدمات
- صفحه‌های درباره ما، تماس و سوالات متداول
- صفحات خدمات، تیم، بلاگ و تحقیقات
- صفحه‌های فروشگاه، سبد خرید، پرداخت و تک‌صفحه‌ی محصول
- پشتیبانی از چند نسخه‌ی زبانی و جهت‌نمای متن

## ویژگی‌های اصلی

- ساختار چندصفحه‌ای و منظم
- طراحی مدرن و مناسب برای نمایش برند و خدمات
- پشتیبانی از نسخه‌های انگلیسی و فارسی
- پشتیبانی از حالت راست‌به‌چپ و چپ‌به‌راست
- استفاده از HTML و Tailwind CSS برای ساخت و استایل‌دهی
- امکان توسعه‌ی آسان به سمت سایت داینامیک یا پنل مدیریتی

## چگونه این پروژه ساخته شده است

این پروژه با استفاده از فایل‌های HTML سازمان‌دهی شده و در پوشه‌ی اصلی، ساختار صفحات مختلف برای نسخه‌های مختلف زبان و چیدمان نگهداری شده است. برای استایل‌دهی نیز از Tailwind CSS استفاده شده و وابستگی‌های مربوطه در فایل package.json ثبت شده‌اند.

ساختار کلی پروژه به این صورت است:

- HTML/labto/English/... : نسخه‌های انگلیسی سایت
- HTML/labto/Farsi/... : نسخه‌های فارسی سایت
- package.json : مدیریت وابستگی‌های پروژه

## چطور پروژه را اجرا کنیم

از آنجا که این پروژه در حالت استاتیک طراحی شده، برای مشاهده‌ی اولیه کافی است فایل HTML مربوط به صفحه‌ی موردنظر را در مرورگر باز کنید. برای مثال:

- HTML/labto/English/v1.2/default/index.html
- HTML/labto/Farsi/rtl/index.html

اگر بخواهید در آینده روی استایل‌ها یا ابزارهای ساخت پیشرفته‌تر کار کنید، می‌توانید وابستگی‌ها را با دستور زیر نصب کنید:

```bash
npm install
```

## مسیر آینده‌ی توسعه

برای ادامه‌ی این پروژه در آینده، این موارد پیشنهاد می‌شود:

- تبدیل پروژه به یک وب‌سایت داینامیک با React یا Next.js
- اتصال به بک‌اند و دیتابیس برای فرم‌های تماس، رزرو و سفارش
- افزودن پنل مدیریت برای ویرایش محتوا
- بهبود سئو، سرعت و تجربه کاربری
- اضافه‌کردن بخش‌های آموزشی، اخبار، پشتیبانی و حساب کاربری

## مشارکت و توسعه

اگر مایل به همکاری هستید، می‌توانید این پروژه را fork کنید و تغییرات خود را با Pull Request ارسال کنید. پیشنهاد می‌شود قبل از تغییرات بزرگ، ساختار صفحات و نسخه‌های زبان را بررسی کنید تا هماهنگی میان نسخه‌ها حفظ شود.
